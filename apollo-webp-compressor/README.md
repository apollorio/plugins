# Apollo WebP Compressor

A WordPress plugin to automatically convert uploaded images to WebP format.

## Features

- Automatically converts JPEG and PNG images to WebP on upload.
- Stores WebP versions in a separate directory.
- Uses the WebPConvert library for conversion.

## Installation

1. Upload the plugin files to `/wp-content/plugins/apollo-webp-compressor/`.
2. Activate the plugin through the 'Plugins' screen in WordPress.

## Requirements

- WebPConvert library (included in the main project's vendor directory).

## Usage

Upload images as usual. WebP versions will be created automatically.

## Based on WebP Express

This plugin is based on the conversion logic from [WebP Express](https://github.com/rosell-dk/webp-express).</content>
